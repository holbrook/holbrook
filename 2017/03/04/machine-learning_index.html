《机器学习》

Posted on 2017-03-04 Edited on 2017-11-02 In 机器学习

《机器学习》, 作者: (美)Tom Mitchell ，译者: 曾华军 / 张银奎 / 等
读书笔记目录。

机器学习关注的问题是:计算机程序如何随着经验积累自动提高性能。

本书展现机器学习中核心的算法和理论。书中的一些算法的实现和数据可以在互联网上^1找到。
